# NxtWatch - Video Streaming Web App

A responsive video streaming application built with React.js. It includes authentication, protected routes, API integration, video playback, saved videos, search, and dark/light theme support.

## Tech Stack
- React.js
- JavaScript
- CSS
- React Router
- REST APIs
- JWT Authentication
- React Context API

## Features
- Login authentication with JWT token
- Protected routes
- Home, Trending, Gaming, Saved Videos, and Video Details pages
- Search videos using API query
- Video player using React Player
- Like, Dislike, and Save video functionality
- Dark and light theme toggle
- Loading, failure, and empty views
- Logout confirmation popup
